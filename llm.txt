# GitMe
> A premium dashboard and AI-powered assistant for analyzing GitHub developer impact.

GitMe provides a more detailed, fair, and comprehensive view of a developer's open-source journey by extracting narratives from raw data—highlighting organizations influenced, languages mastered, and specific impact of contributions.

## Key Features
- **Detailed Work Synthesis**: Fetches full profile metadata and personal READMEs.
- **Granular Contribution Discovery**: Categorized views for Pull Requests, Issues, and Discussions.
- **Multi-Year Activity Context**: Multi-year contribution calendar for growth tracking.
- **Technical Stack Visualization**: Automated extraction of primary technologies and collaborations.
- **Integrated CV Viewer**: Premium modal for viewing professional resumes (PDF).
- **GitMe AI Assistant**: Persistent technical recruiter persona for on-demand profile analysis.

## Technical Stack
- **Framework**: React 18 with Vite 7.
- **Styling**: Tailwind CSS 3.4 with custom GitHub-themed color palette.
- **Icons**: Lucide React.
- **APIs**: GitHub GraphQL API (v4), OpenRouter.
- **Markdown**: `react-markdown` with GFM and Rehype Raw.
